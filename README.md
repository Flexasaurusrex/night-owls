## Quick Setup

1. Clone this repo
2. Run `npm install`
3. Get free Foursquare API key at [foursquare.com/developers](https://foursquare.com/developers)
4. Add `NEXT_PUBLIC_FOURSQUARE_API_KEY=your_key_here` to `.env.local`
5. Run `npm run dev`
6. Deploy to Vercel

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Foursquare Places API
- Lucide React Icons
